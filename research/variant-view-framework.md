# Variant-View Fundamental Framework (Gap 2)

**Purpose:** move from *consuming* consensus (analyst PTs, ratings) to *forming a differentiated, quantified view* — the part of fundamental work where institutional alpha actually lives. Run this on any name we're sizing a real position in, or on any tracked name with an earnings catalyst inside the window.

**Sourcing:** verified outlets ONLY (Reuters, Bloomberg, CNBC, WSJ, AP, MarketWatch, FT, Barron's, company IR, exchange/Nasdaq). Prices labeled web-approx; authoritative price = live TV feed. Never forums/Reddit/Stocktwits/SA comments.

## The template — fill every row
| # | Field | What to capture |
|---|-------|-----------------|
| 1 | **Consensus snapshot** | This-Q + next-Q + FY EPS/revenue consensus; company guide; rating distribution (buy/hold/sell counts); mean & median PT; current px (web-approx). |
| 2 | **Estimate-revision trend** | Direction + magnitude of EPS/rev revisions over ~30d and ~90d (rising/falling/stable). List notable PT changes last ~2 wks (analyst, old→new). *Revisions are the single most predictive fundamental signal — trade with the revision trend, not against it.* |
| 3 | **Options-implied move** | Market-implied move into the next print (ATM straddle / IV), vs the stock's typical post-earnings move. Source it. |
| 4 | **Key debate / variant axis** | The 1–2 sentence bull-vs-bear crux for the catalyst (what the print actually turns on). |
| 5 | **OUR VARIANT VIEW** | Where we sit vs consensus and WHY — one differentiated, defensible point grounded in rows 1–3. State the asymmetry (skew up/down) + what confirms/refutes it. |
| 6 | **Risk flags** | Valuation (fwd P/E vs growth / PEG), notable short interest, binary overhang. |

## How to use it
- **The edge is row 5 vs row 1.** If our view = consensus there's no trade, only beta. The setups worth sizing are where revisions (row 2) and/or the implied move (row 3) disagree with where consensus PTs (row 1) sit.
- **The "bar" check:** when revisions have run *ahead* of guidance and the stock is extended, the bar is high → a beat may still sell off (fade/trim setup). When revisions are *rising* but the stock lags and IV is cheap, that's positive asymmetry.
- **Never hold through a binary blind:** every position gets row 3 (implied move) before an earnings/FDA/Fed date so size matches the gap risk. Pairs with the [Positioning & Flow framework](positioning-flow-framework.md) (Gap 3).

## What's repeatable vs gated
- **Repeatable daily, free:** rows 1, 2, 4, 6 from verified outlets; row 3 implied move when a verified source prints it.
- **Gated (would deepen this):** analyst-by-analyst old→new estimate history, real-time consensus aggregation (FactSet/Visible Alpha/Bloomberg), precise live ATM-straddle pricing at our timestamp, channel checks / a built DCF. We approximate the variant view from revision trend + implied move + valuation-vs-growth, which is most of the practical signal.
